# Portfolio Website

A responsive personal portfolio website showcasing my skills, projects, and contact information.

## Project Overview

This portfolio website serves as an online resume and demonstration of my web development skills. It features a clean, modern design with responsive layouts that work well on all devices.

## Features

- **Responsive Design**: Mobile-first approach ensures compatibility with all screen sizes
- **Interactive UI**: Smooth scrolling, animations, and interactive elements enhance user experience
- **Contact Form**: Allows visitors to send messages directly through the website
- **Project Showcase**: Highlights my best work with project descriptions and links
- **Skills Section**: Visual representation of technical and soft skills

## Sections

1. **Hero Section**: An eye-catching introduction with a professional tagline
2. **About Me**: Personal introduction and background information
3. **Skills**: Technical skills with progress bars and soft skills
4. **Projects**: Showcase of completed projects with descriptions and links
5. **Contact**: Form for visitors to reach out and connect

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome (for icons)
- Google Fonts

## Setup and Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/portfolio-website.git
   ```

2. Navigate to the project directory:
   ```
   cd portfolio-website
   ```

3. Open `index.html` in your browser to view the website locally.
